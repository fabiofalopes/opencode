# OpenCode Configuration Knowledge Base

This directory contains accumulated knowledge about OpenCode configuration.

## Files

| File | Purpose |
|------|---------|
| config-errors.md | Error patterns and root causes |
| config-remediations.md | Proven solutions and procedures |

## Usage

The `opencode-config-manager` agent consults these files before making changes.

## Contributing

When you encounter and solve a new configuration issue:
1. Document the error pattern in `config-errors.md`
2. Document the solution in `config-remediations.md`
3. Optionally store in memory MCP for cross-session learning
